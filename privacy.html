Privacy Policy

Effective date: December 2025

This browser extension extracts publicly visible product information from Amazon pages (such as ASINs, titles, prices, variations, images, and availability). All processing happens locally in the user's browser. This extension does not collect, store, transmit, or share any personally identifiable information.

Information We Collect

None. The extension does not collect any personal information, browsing history, authentication data, financial information, or sensitive data of any type.

How Data Is Processed

  • All product extraction and analysis occurs locally in the user's browser.
  • No data is transmitted to external servers.
  • Temporary settings and collected product data are stored only in chrome.storage.local.

Permissions and Use

  • activeTab — Reads product details from the Amazon page the user is viewing.
  • scripting — Executes scripts to extract ASINs and product variations.
  • storage — Stores temporary extension settings and data locally.
  • downloads — Allows CSV export of collected product information.
  • clipboardWrite — Lets the user copy extracted data to the clipboard.
  • alarms — Manages timing during automated ASIN availability checks.
  • host permission for https://*.amazon.com/ — Needed to access publicly visible Amazon product pages.

Data Sharing

This extension does not share, sell, or transfer any user data to third parties. No analytics or tracking systems are used.

Children's Privacy

This extension is not intended for children and does not knowingly collect any data from individuals under 13.

Changes to This Policy

If the extension's functionality changes and begins collecting or transmitting user data, this policy will be updated accordingly.
